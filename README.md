# Prophet-Model-for-Predict-Tourism-Climate-Index-TCI-in-Bali
It's provided data structure, analysis concept, data cleaning, and forecasting the TCI Index for the next 2 years and 2 month in Bali Island

# Requirements
- Python 3.5.x
- Pystan 2.18
- Prophet Model
- Pmdarima
- Statsmodel
- Matplotlib
- Seaborn
